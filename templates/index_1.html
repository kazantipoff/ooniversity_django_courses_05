Python&Django by PyBursa

Description

Курс Python&Django by PyBursa расчитан на пользователей владеющих начальными навыками разарботки на языке Python и предполагает наличие у студента базовых знаний языка.Покрывает основной синтаксис и инструкции языка, некоторые встроенные модули, основы ООП.

Program

Весь курс разделен на шесть недель: первая неделя это html, css и GIT. Вторая неделя будет посвящена Django. Третья неделя это структура Web-приложений и работа с базами данных, четвертая неделя - кастомизация админки и язык шаблонов, пятая неделя будет проведена за изучениями форм, типов полей, виджетов и аттрибутов валидации. На шестой, заключительной неделе, пройдет реализация стандартных задач, проведение их отладки и тестирование.

Perspectives

Успешное окончание курса предполагает подготовку студента на уровне достаточном для получения вакансии Junior-а в любой компании занимающейся разработкой ПО. По статистике 97% обучающихся получают указанные должности в течении первых 3 месяцев после окончания курса.
